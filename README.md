# ΚΠΓ Γαλλικά - GitHub Pages site

Στατικό site για εξάσκηση στα θέματα ΚΠΓ Γαλλικών. Περιλαμβάνει επίσημους συνδέσμους θεμάτων, ενσωματωμένα PDF/MP3 για τα διαθέσιμα θέματα, εκφωνήσεις μέσα στο φύλλο συμπλήρωσης, φίλτρα ανά επίπεδο, επιλογή ενός θέματος κάθε φορά, χώρο απαντήσεων, σημειώσεις, πρόοδο, εισαγωγή και εξαγωγή δεδομένων.

## Αρχεία

- `index.html`
- `styles.css`
- `app.js`
- `paper-assets.js`
- `paper-prompts.js`
- `assets/exams/`
- `.nojekyll`

## Δημοσίευση σε GitHub Pages

1. Δημιούργησε ένα νέο repository στο GitHub.
2. Ανέβασε τα αρχεία του φακέλου στη ρίζα του repository.
3. Άνοιξε `Settings` → `Pages`.
4. Στο `Build and deployment`, διάλεξε `Deploy from a branch`.
5. Επίλεξε branch `main` και folder `/root`.
6. Το site θα ανοίγει σε διεύθυνση τύπου `https://USERNAME.github.io/REPOSITORY/`.

## Πηγές θεμάτων

- Επίσημη σελίδα ΚΠΓ Γαλλικών: https://rcel2.enl.uoa.gr/kpg/gr_past_papers_fr.htm
- Συνδεδεμένα αρχεία/σελίδες ΥΠΑΙΘΑ και ΑΠΘ όπως εμφανίζονται στην επίσημη λίστα.
- Για αρχεία που δεν είναι πια διαθέσιμα στον τρέχοντα σύνδεσμο ΥΠΑΙΘΑ, χρησιμοποιήθηκαν αποθηκευμένα αντίγραφα του ίδιου επίσημου URL από το Internet Archive.

## Σημείωση

Οι απαντήσεις αποθηκεύονται τοπικά στον browser. Για μεταφορά σε άλλη συσκευή χρησιμοποίησε την εξαγωγή JSON και μετά την εισαγωγή.
